# SimpleLinearRegression

Linear regression from stratch using only Numpy

Required libraries

- Numpy
- Matplotlib
- Sci-kit learn
- tqdm
- JupyterLab

## installation

### Classic

```bash
pip install -r requirements.txt
jupyter lab
```

### Portable

#### Manual

```bash
python -m venv venv
source venv/{bin,Scripts}/activate
pip install -r requirements.txt
jupyter lab
```

#### Automated

```bash
./helper.sh install
./helper.sh launch
```
